## Design System

### Colors Used
These are the core colors used across the UI (from the CSS variables and base styles):

- Backgrounds: `#fbfbfa`, `#f4f5f4`
- Text: `#09080a`, `#000000`, `#f4f5f4`
- Accents: `#1d4c56`, `#45bc97`, `#96b099`, `#0c8c5e`
- Supporting tones: `#b4ab48`, `#88600e`, `#d1d092`, `#3a88c8`, `#bca794`
- UI neutrals: `#08090acc`

### Typography
- Primary font: `Inter`, sans-serif
- Accent/mono font: `Geist Mono`, monospace (used for small headings)

## Clone the Repository

```bash
git clone https://github.com/Ritam369/Mintlify_Landing.git
cd Mintlify_Landing
```

## Run Locally

**Option 1: Using Live Server (Recommended)**
- Install [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
- Right-click on `index.html` and select "Open with Live Server"

**Option 2: Direct Browser Opening**
- Simply double-click `index.html` to open it in your default browser
